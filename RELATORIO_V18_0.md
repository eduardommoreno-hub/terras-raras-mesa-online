# v18.0 — Remaster Funcional da Floresta Negra

Base: v17.3 online estável.

## Objetivo
Iniciar a implementação visual oficial da campanha pelo primeiro mapa, na ordem correta da Jornada.

## Implementado
- Floresta Negra recebeu o primeiro layout visual cinematográfico funcional.
- Imagem remasterizada aplicada como camada visual do mapa.
- Token da jogadora reduzido para formato pequeno de caminhada, sobre a trilha.
- Hotspots preservam os IDs internos do código.
- Botões visuais funcionais no rodapé: Inventário, Cartas, Mapa, Missões e Diário.
- Atalhos no painel do local para Cartas, Missões, Diário e Jornada.
- Os demais mapas continuam no layout anterior, aguardando remaster mapa por mapa.

## Regra preservada
O código continua sendo a fonte de verdade para IDs, missões, cartas, eventos e progresso. A imagem define o padrão visual.
