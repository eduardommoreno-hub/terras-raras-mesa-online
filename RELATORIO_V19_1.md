# Terras Raras v19.1 — Floresta Negra Premium Map Only Funcional

## Objetivo
Usar o mapa premium `floresta_negra_v18_map_only.png` como base visual da Floresta Negra, mantendo todos os elementos funcionais renderizados por código.

## Correções
- Substituído o SVG verde simplificado pelo mapa cinematográfico premium.
- Hotspots invisíveis/funcionais posicionados sobre os locais reais da arte.
- Token único renderizado por código, sem token embutido no mapa-base.
- Painel direito mantido dinâmico: personagem, stats, nível e cartas privadas.
- Barra inferior preservada em HTML/CSS, funcional e no padrão dourado.
- Layout geral da Floresta reforçado para o mesmo padrão visual premium.

## Regra
Imagem = mapa-base premium.
Código = token, hotspots, painel, cartas, stats e botões.
