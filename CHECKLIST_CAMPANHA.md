# Checklist de Auditoria — Campanha Terras Raras

Use este checklist antes de iniciar a fase v18.0.

## Fluxo principal

- [ ] Login funciona.
- [ ] Cadastro novo fica pendente.
- [ ] Admin aprova cadastro.
- [ ] Criação de sala funciona.
- [ ] Entrada por código funciona.
- [ ] Sala de espera funciona.
- [ ] Início manual pela Mestre funciona.
- [ ] Jogadora escolhe personagem.
- [ ] Chat funciona.
- [ ] Chat de bastidores funciona para staff.
- [ ] IA local recebe pedidos via worker.

## Jornada completa

- [ ] Zona 1 — Floresta Negra abre corretamente.
- [ ] Zona 2 — Fábrica dos Doces Pesadelos abre corretamente.
- [ ] Zona 3 — Cidade dos Relógios Parados abre corretamente.
- [ ] Zona 4 — Montanhas Arcaicas abre corretamente.
- [ ] Zona 5 — Gelo Eterno abre corretamente.
- [ ] Zona 6 — Alexandria abre corretamente.
- [ ] Zona 7 — Tempestade dos Deuses abre corretamente.
- [ ] Zona 8 — Correr ou Morrer abre corretamente.
- [ ] Zona 9 — O Vazio abre corretamente.
- [ ] Cada abertura cinematográfica aparece antes do mapa.
- [ ] Cada botão “Começar aventura” leva ao mapa certo.
- [ ] Cada portal avança para a zona correta.

## Sistemas da Mestre

- [ ] Central da Mestre carrega.
- [ ] Modo condução abre.
- [ ] Eventos da Mestre salvam e exibem.
- [ ] Cartas da Aventura são enviadas.
- [ ] Cartas podem ser guardadas.
- [ ] Cartas guardadas aparecem no inventário/diário.
- [ ] Biblioteca da Mestre abre nas zonas com conteúdo.
- [ ] Biblioteca gera evento, carta, item e prompt de IA.
- [ ] Missões mostram objetivos corretos da zona atual.
- [ ] Diário Visual gera linha do tempo.
- [ ] Resumo da sessão pode ser gerado e salvo.

## Operação da sala

- [ ] `/kick` remove jogadora da sala.
- [ ] Remoção libera vaga.
- [ ] Mestre/admin não é removida indevidamente.
- [ ] Inventário adiciona item por rota atômica.
- [ ] Inventário remove item por rota atômica.

## Segurança infantil

- [ ] Chat bloqueia conteúdo impróprio.
- [ ] Cartas passam pelo filtro.
- [ ] Eventos da Mestre passam pelo filtro.
- [ ] Notas/textos enviados às jogadoras passam pelo filtro.
- [ ] Logs aparecem somente quando há bloqueio.

## Dispositivos

- [ ] Desktop.
- [ ] Tablet.
- [ ] Celular.
- [ ] Painel lateral abre/fecha no mobile.
