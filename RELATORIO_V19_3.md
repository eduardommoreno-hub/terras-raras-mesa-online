# v19.3 — Design System Global Premium Terras Raras

## Objetivo
Padronizar a identidade visual premium desde o login, hub, criação de sala, entrada por código, painéis, botões, inputs, cards e área de jogo.

## Alterações
- Criado bloco global de CSS v19.3 com design system premium.
- Botões principais com gradiente dourado, brilho, borda ornamental e ícone discreto.
- Botões secundários escuros com borda dourada fina.
- Inputs, selects e textareas padronizados com fundo escuro, borda dourada e foco premium.
- Login/cadastro recebeu acabamento cinematográfico.
- Hub recebeu painéis, títulos, cards e formulários no mesmo padrão.
- Painéis internos, cards, notas, chat e modais receberam o mesmo padrão visual.

## Observação
Esta versão não altera a lógica de tokens, permissões da Ajudante, anotações ou WebSocket. É uma camada visual global para unificar o produto.
